Политика обработки персональных данных

Редакция от 11 июня 2026 года

1. Общие положения

Настоящая Политика определяет порядок обработки персональных данных Пользователей сервиса «ИИ-Юрист» в соответствии с Федеральным законом от 27.07.2006 № 152-ФЗ «О персональных данных».

Оператором персональных данных является ООО «ИИ-Юрист» (далее — «Оператор»).

Используя Сервис, Пользователь даёт согласие на обработку своих персональных данных в соответствии с настоящей Политикой.

2. Какие данные мы собираем

Категория данных Конкретные данные Цель сбора
Идентификационные данные Имя, фамилия, отчество, имя пользователя Идентификация пользователя в системе
Контактные данные Адрес электронной почты Авторизация, уведомления
Данные учётной записи Хэш пароля Аутентификация (пароль хранится в зашифрованном виде)
Данные диалогов Текст сообщений, история чатов Предоставление услуг, улучшение качества агентов
Сгенерированные документы DOCX-файлы исков и претензий Хранение результатов работы для скачивания
Технические данные IP-адрес, user-agent, время запросов Аудит безопасности, защита от мошенничества

3. Цели обработки персональных данных

  • предоставление доступа к функциям Сервиса;
  • генерация юридических документов на основании данных Пользователя;
  • обеспечение безопасности учётных записей;
  • улучшение качества работы AI-агентов;
  • соблюдение требований законодательства РФ.

4. Основания для обработки

Обработка персональных данных осуществляется на основании:

  • согласия Пользователя (ст. 9 Федерального закона № 152-ФЗ);
  • исполнения договора, стороной которого является Пользователь (ст. 6 ч. 1 п. 5);
  • соблюдения требований законодательства (ст. 6 ч. 1 п. 2).

5. Срок хранения данных

Персональные данные хранятся в течение всего времени использования Сервиса, а также в течение 3 лет после удаления учётной записи — в целях соблюдения требований законодательства об аудите и логировании.

Сгенерированные документы хранятся до их удаления Пользователем или до удаления учётной записи.

6. Передача данных третьим лицам

Оператор не передаёт персональные данные третьим лицам, за исключением:

  • случаев, предусмотренных законодательством РФ (по запросу уполномоченных органов);
  • технических партнёров, обеспечивающих работу инфраструктуры (в объёме, необходимом для оказания услуг, с соблюдением конфиденциальности).

Для генерации ответов используется модель GigaChat (ПАО «Сбербанк»). Тексты сообщений могут передаваться в API GigaChat для обработки. Политика конфиденциальности Сбербанка доступна на сайте developers.sber.ru.

7. Права пользователя

В соответствии с Федеральным законом № 152-ФЗ Пользователь вправе:

  • получить информацию об обработке своих персональных данных;
  • потребовать уточнения, блокирования или уничтожения своих данных;
  • отозвать согласие на обработку персональных данных;
  • удалить учётную запись через настройки профиля.

Для реализации прав обращайтесь на адрес: support@ai-jurist.ru

8. Защита данных

Оператор принимает следующие меры для защиты персональных данных:

  • хранение паролей в виде хэша (bcrypt);
  • передача данных по протоколу HTTPS;
  • ограничение доступа к базе данных (закрытые порты);
  • аудит-логирование всех действий пользователей;
  • регулярное резервное копирование.

9. Cookies

Сервис использует localStorage браузера для хранения JWT-токенов авторизации. Данные хранятся только на устройстве Пользователя и не передаются третьим лицам.

10. Изменение политики

Оператор вправе вносить изменения в настоящую Политику. Актуальная версия доступна по ссылке на странице регистрации. При существенных изменениях Пользователь будет уведомлён.
